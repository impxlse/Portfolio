Mes Projets

Conception et mise en œuvre d’une infrastructure réseau sécurisée

Dans ce projet encadré par notre BUT, notre équipe de trois personnes était chargée de concevoir et dimensionner une infrastructure réseau robuste, entièrement nouvelle, destinée à être déployée dans un environnement similaire à celui d'une entreprise. Les contraintes étaient multiples, notamment l'utilisation exclusive de logiciels open source et la création d'un réseau comprenant des éléments tels qu'un routeur/firewall, un serveur DNS, un serveur DHCP, ainsi que des services Kerberos/LDAP, entre autres. Nous avons dû produire des livrables comprenant des scripts permettant de déployer les machines virtuelles cruciales telles que le routeur et le serveur DNS. Ce processus garantissait la reproductibilité et la facilité de déploiement de l'infrastructure.

Compétences : conception d'infrastructure réseau, sécurité réseau, Documentation sur les logiciels Open Source utilisés, création et configuration de machines virtuelles, scripts de déploiement, gestion d'une équipe.

Packaging d'une application

Dans le cadre de ce projet, ma responsabilité consistait à déployer l'application open source "Leed RSS", un agrégateur de flux RSS. L'objectif principal était d'automatiser ce déploiement en utilisant Docker, afin de rendre l'application rapidement disponible via l'utilisation de Docker-compose. Parallèlement, il était essentiel de gérer les dépendances de l'application, notamment :

  • Configuration d'un serveur Apache (recommandé)
  • Installation de PHP avec une version minimale de 7.2
  • Mise en place d'une base de données MySQL

Compétences : Docker, Docker-compose, Dockerfile, Apache, PHP, MySQL

Déploiement LAPS sur les serveurs Entreprise

Dans le cadre de mon projet relatif à LAPS (Local Administrator Password Solution) au cours de mon alternance à LYNRED, j'ai travaillé sur l'implémentation de cette solution Microsoft. L'objectif était de renforcer la sécurité des mots de passe des administrateurs locaux en les stockant de manière sécurisée dans Active Directory. Le processus a impliqué une analyse méticuleuse des implications de LAPS, notamment en ce qui concerne la compatibilité avec les serveurs Windows et l'impact potentiel sur les services et tâches planifiées exécutés par ces serveurs. Cette analyse a été menée à l'aide de scripts PowerShell, permettant ainsi une visibilité approfondie des risques potentielles.

Compétences : Active Directory, PowerShell, gestion des serveurs Windows
